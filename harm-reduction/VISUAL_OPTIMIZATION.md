# G-Timer 視覺優化記錄

## 優化目標
1. 調整呼吸球與數字之間的視覺層次（避免「疊在一起」）
2. 整體佈局垂直置中（避免偏上）
3. 確保數字清晰可讀，呼吸球作為背景氛圍

## 實施方案

### 1. 數字層（主要資訊）
- **字體大小**：3rem → 3.5rem
- **letter-spacing**：2px → 3px
- **text-shadow**：新增 `0 0 20px rgba(255, 255, 255, 0.3)` 柔和光暈
- **定位**：保持 `position: absolute` + `transform: translate(-50%, -50%)`，確保不受動畫影響

### 2. 呼吸球層（背景氛圍）
- **尺寸**：220px → 280px（增加視覺空間）
- **不透明度**：降低動畫範圍（0.6-0.9，原 0.8-1.0）
- **模糊效果**：新增 `filter: blur(1px)` 製造霧狀質感
- **邊框**：2px solid → 1.5px，透明度降低到 0.3
- **背景漸層**：調整為 `rgba(255, 45, 149, 0.15) → 0.05 → 0`，更柔和
- **動畫幅度**：scale(0.92-1.08)，增加呼吸感

### 3. 整體佈局
- **垂直置中**：`.timer-container` 的 `min-height: 80vh` → `100vh`
- **間距**：呼吸球 `margin-bottom: 30px` → `40px`

### 4. 狀態色彩優化
- **紅色（高峰期）**：
  - 邊框透明度 0.4
  - 背景漸層 `rgba(255, 0, 68, 0.15) → 0.05 → 0`
  - box-shadow 強度增加到 40px
- **金色（警戒期）**：
  - 邊框透明度 0.4
  - 背景漸層 `rgba(255, 215, 0, 0.12) → 0.04 → 0`
- **白色（安全期）**：
  - 邊框透明度 0.3
  - 背景漸層 `rgba(243, 244, 245, 0.08) → 0.02 → 0`

## 視覺效果
- ✅ 數字清晰可讀，作為主要資訊焦點
- ✅ 呼吸球柔和模糊，作為背景氛圍
- ✅ 整體佈局垂直置中，視覺平衡
- ✅ 文字不隨球體動畫縮放，避免暈眩

## 技術細節
- 使用 `filter: blur(1px)` 而非更大的值，避免過度模糊
- 保持 `pointer-events: none` 在文字層，確保不干擾互動
- 動畫使用 `ease-in-out`，呼吸節奏自然
